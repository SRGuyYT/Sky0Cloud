Reliable Config Delivery

Element can load both default and domain-specific configuration files, avoiding startup failures and the “Couldn’t load page” flow.

Config: /config.json and /config.sky0cloud.dpdns.org.json

Static Assets Always Served

Welcome page, service worker, and favicon are mounted directly into nginx document root to prevent missing-file errors across restarts.

Assets: /welcome.html, /sw.js, /favicon.ico

Proxy + Matrix Routing

Caddy serves Element at root and forwards Matrix API traffic to Conduwuit with headers suitable for Cloudflare fronting.

Homeserver: sky0cloud.dpdns.org
